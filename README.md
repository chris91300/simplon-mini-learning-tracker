# MINI LEARNING TRACKER

## Pour lance le frontend:
```bash
cd frontend
npm run dev
```

## Pour lance le backend:
dans le dossier backend/src/main/resources,

créé un fichier 'application.properties'.

A l'intérieur, mets tes variables comme indiqué dans le 'application.example.properties'

ensuite:
```bash
cd backend
./mvnw spring-boot:run
```
